Sobre

Dashboard Covid19

Este Dashboard da Covid19 fornece uma viso geral do novo surto do novo Coronavrus COVID-19 em Cabo Verde. Este dashboard foi construdo com R usando a estrutura R Markdown.

Autoria

O dashboard foi desenvolvido e mantido por Mrio Cardoso.

Dados

Os dados de entrada para este dashboard so os pacotes R covid19cvdata e coronavirus R (verso de desenvolvimento). Os dados e o dashboard so atualizados diariamente.

Fonte de dados

Os dados brutos do pacote covid19cvdata so obtidos do website covi19.cv e o pacote coronavrus do Centro de Cincia e Engenharia da Universidade Johns Hopkins (JHU CCSE).

Pacotes

  • Interface do dashboard - o pacote flexdashboard.
  • Visualizao - o pacote plotly para os grficos e mapview para o mapa
  • Manipulao de dados - dplyr e tidyr
  • Tabelas - o pacote DT
