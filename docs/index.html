Sobre

Dashboard Covid19CV

Este Dashboard pretende fornecer uma viso geral do surto do COVID-19 em Cabo Verde.

O dashboard foi construdo com R com a estrutura R Markdown, utilizando os dados das fontes oficiais de Cabo Verde e da Johns Hopkins University disponvel no pacote R coronavirus.

Autoria

O dashboard foi desenvolvido e mantido por Mrio Cardoso.

Este projeto pessoal e utiliza dados pblicos

Dados

Os dados de entrada para este dashboard so os pacotes R covid19cvdata e coronavirus (ambas em verso de desenvolvimento). Os dados e o dashboard so atualizados diariamente.

Fonte de dados

Os dados brutos do pacote R covid19cvdata so obtidos do dashboard oficial dos dados do Covid19 em Cabo Verde, covi19.cv e os dados do pacote coronavirus do Centro de Cincia e Engenharia da Universidade Johns Hopkins (JHU CCSE).

Pacotes

  • Interface do dashboard - o pacote flexdashboard.
  • Visualizao - o pacote plotly para os grficos e mapview para o mapa
  • Manipulao de dados - dplyr e tidyr
  • Tabelas - o pacote DT
